# Adidas Sales Dashboard Analysis

## Project Overview
This project analyzes Adidas sales data using Microsoft Excel.
The dashboard provides insights into sales performance, profit, and regional trends.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Power Query
- Advanced Formulas (XLOOKUP, INDEX-MATCH)

## Features
- Interactive dashboard using slicers
- Region-wise and product-wise analysis
- Profit and revenue calculation
- Data cleaning using Power Query

## Skills Demonstrated
- Data Analysis
- Excel Dashboard Development
- Data Cleaning
- Business Reporting
